Example One

The new Bootstrap grid system applies to mobile first. When you declare a specific grid size, that is the grid for that size and above. This can be a little hard to grasp at first so here’s an example. Since the grid system now cascades up from mobile devices, this is how this code will look.

    For example, let’s say you want a site that has:
  • (1) Columns on Extra small devices - Phones (less then 768px)
  • (2) Columns on Medium devices - Tablets and Laptops (≥992px)
  • (4) Columns on Large devices - Desktops (≥1200px)
